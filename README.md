## Как пользоваться

Инструкция

```
$ git clone

# Установите все зависимости
$ npm install (можно и Yarn)

# Запуск watch-режима (hot-reload):
npm start

# Можно сбилдить всё:
npm run build

# Минифицировать всё:
npm run build.all

# Юнит тесты:
npm run test
```
